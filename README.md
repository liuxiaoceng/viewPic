ViewPic -- 一个简单的本地图片浏览工具  

## 介绍  

这是一个简单的本地图片浏览工具，不需要借助任何工具，只要你有浏览器就可以使用  
它已经完成以下功能 ：  

1. 查看图片  
2. 预览图片：底部的图片列表可以预览图片  
3. 点击切换上一张和下一张图片  
4. 点击更新图片列表的效果  
3. 选择文件夹，更新图片列表    
4. 逆时针旋转查看图片，顺时针旋转查看图片  

![viewPic - 截图](images/viewPic.png)  

## 实现技术  

1. HTML  
2. CSS
3. 纯 javascript

## 使用  

下载，双击 `index.html` 即可
